# weather-checker-cc-pptr

Weather checker with CodeceptJS and Puppeteer

## Requirements

- Node

## How to install?

- Clone this project.
- Step into the project directory.
- Run: `npm install`.

## How to run?

- Step into the project directory.
- Run: `npm test`.

## Where is the test file?

- `./tests/search_test.js`

## What are the test steps?

- Opens the homepage,
- Accepts the cookies.
